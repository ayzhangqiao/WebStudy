Error code Note How to handle it
E1 The main supply voltage is too high Check the main power supply
E2 The height deviation between two columns exceeds 1cm Re-initialize
E3 The hand controller is not connected. Pick up the control device
E4 Hand Controller communication Error Check the hand controller line
E5 In the event of a drag-stop display Loosen the button and run again
E6 Main power start did not succeed Replace the main power supply, or check the main power cable
E7 Protection when the main power supply is running Power up again
E8 Tilt when the desk is running Re-initialize
E9 High power supply temperature Wait for the power supply temperature to drop and rerun
E11、E21 Motors Not Connected Check the cable connection
E12~17 E22~27 Technology problem Please get in touch with factory
E18、E28 overload Reduces load
E40 Controller line dropped Check the cable
E41 There is an error in the concatenation signal Check the cable, or replace the controller
E42 Memory error Replacing the Controller
E43 Disable sensor error Replacing the Controller
